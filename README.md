# Loan Approval Prediction

## Project Overview
This project predicts whether a loan application will be approved using Machine Learning.

## Dataset
Loan Approval Prediction Dataset

## Steps Performed
- Data Loading
- Data Cleaning
- Handling Missing Values
- Encoding Categorical Variables
- Train-Test Split
- Logistic Regression Model
- Model Evaluation

## Results
The model achieved good accuracy in predicting loan approvals.

## Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- Jupyter Notebook
